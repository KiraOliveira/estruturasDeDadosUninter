## Atividade prática da matéria de "Estrutura de Dados" da Uninter:

Repositório destinado ao armazenamento e organização dos códigos desenvolvidos para a atividade prática, abordando conceitos fundamentais de gerenciamento de memória e performance.

- Implementação de Lista Encadeada Simples para triagem hospitalar.
- Implementação de Tabela Hash com Endereçamento em Cadeia para sistema de emplacamento.

* Utilizando a linguagem [Python](https://www.python.org/).

---

## Sobre o repositório:

1 - Sistema de Triagem: Criado para realizar o gerenciamento de uma fila de atendimento hospitalar, onde os pacientes cadastrados com cartões Amarelos (A) possuem prioridade sobre os Verdes (V), utilizando lógica de inserção ordenada em lista encadeada.

2 - Sistema de Emplacamento (Hash): Criado para organizar uma tabela de 10 posições com os estados Brasileiros cadastrados, utilizando uma função baseada em valores ASCII e tratando colisões através de listas encadeadas.

---

## Tecnologias Utilizadas

- Python 3.x
- Estruturas de Dados: Listas Encadeadas e Tabelas Hash
- Lógica de Ponteiros: Manipulação de referências de memória (next/proximo)
- VS Code
